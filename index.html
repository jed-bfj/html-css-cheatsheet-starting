1. Doctype Declaration

Pretty straightforward actually - you need to declare:

<!DOCTYPE html/>
...
</html>
at the beginning of your html file.

2. General Structure

Within html brackets, establish a head, body, and footer.

<head>

...

</head>
<body>

...

</body>
<footer>

...

</footer>
head is where you place link tags, to bring in javascript and css files (or other dependencies like SDKs)
body is where you place the meat of the content
footer is self explanatory

3. Attributes, Classes, ID's

Each html element can be assigned attributres, which modifies or extends their behavior:

<link href="css/style.css" rel="stylesheet">
<div id='this-is-an-id></div>
<p class='content'></p>
<p style='color: blue; font-size: 16px;'></p>
class allow you to group similar components for easy and modular modifications (CSS, javascript)
id is more specific and target individual elements
CSS styles can be added inline, though it's best practice to have a separate css file (more on that later)

4. Links & Anchors

link and anchor a tags are different.
link tags are primarily used in the head section and link to external resources, like stylesheets and icons and fonts:

<link href="css/style.css" rel="stylesheet">
rel defines the relationship between current document and linked resource, href specifies url of linked soure
anchor a tags are used to create hyperlinks that navigate users elsewhere, externally or on the same page:
<a href="https://www.theringer.com">
</a>
Click this link to go to theringer.com
Unlike link tags, anchor a tags must be closed (they are not self-closing)
href defines the url path for the linked resource
The target attribute, set to '_blank' will open a new tab vs linking the current one

5. Images & Videos

Images and videos can be added like so:

<img src="image.jpg" alt="Description of Image>
<video controls width='400' height='300'>

<source src='video.mp4' type='video.mp4'>

Your browser does not support the video tag.

</source>

</video>
image is a self-closing tag, video is not
video requires a source tag to indicate src and file type
src is the url or filepath for both
alt provides alternative text to display when image cannot be; video will display the text between the tags
controls displays video controls like pause, play etc
width and height set dimensions

6. Tables

Tables can be a bitch, especially to style and format correctly, so I'll just outline the core code and explain:

<table>

<thead>

<tr>

<th scope='col'>Company Name</th>

<th scope='col'>Number of Items to Ship</th>

<th scope='col'>Next Action</th>

</tr>

</thead>

<tbody>

<tr>

<td>Joey's cool company</td>

<td>69</td>

<td>Baileys</td>

</tr>

...

</tbody>

<tfoot>

<tr>

<td>Total</td>

<td>420</td>

</tr>

</tfoot>

</table>
Company Name Number of Items to Ship Next Action
Adam's Greenworks 14 Package Items
Davie's Burgers 2 Send Invoice
Baker's Bike Shop 3 Send Invoice
Miss Sally's Southern 4 Ship
Summit Resort Rentals 4 Ship
Strike Fitness 1 Enter Order
Total 28
The table element creates a table.
The tr element adds rows to a table.
To add data to a row, you can use the td element.
Table headings clarify the meaning of data. Headings are added with the thelement.
Table data can span columns using the colspan attribute.
Table data can span rows using the rowspan attribute.
Tables can be split into three main sections: a head, a body, and a footer.
A table’s head is created with the thead element.
A table’s body is created with the tbody element.
All the CSS properties you learned about in this course can be applied to tables and their data.
A table’s footer is created with the tfoot element.
